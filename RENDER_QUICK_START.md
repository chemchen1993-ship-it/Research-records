# Render Quick Start

Use this link to start deployment:

- [https://render.com/deploy?repo=https://github.com/chemchen1993-ship-it/Research-records.git](https://render.com/deploy?repo=https://github.com/chemchen1993-ship-it/Research-records.git)

## In Render

1. Sign in to Render.
2. Open the deploy link above.
3. Confirm the repository and blueprint.
4. Keep the included persistent disk enabled.
5. Start the deploy.

## After deploy finishes

1. Open the generated `https://...onrender.com` address on your computer.
2. Open the same URL in Safari on iPad.
3. Tap `Share` on iPad.
4. Tap `Add to Home Screen`.

## Important

- Saved records sync through Render.
- Unsaved drafts still stay local to each device.
- Do not remove the persistent disk, or saved records can be lost on redeploy.
